"Hello Node IT!!"; Welcome: Node IT is a small and lightweight code editor builtin on Ace and a simple static http server. It's focused on web and webapp development you can programming on the left side and see your app on the right side, or other html page as you configure. Shortcuts: Ctrl+Shift-P, Command+Shift-P => input[0..100] Resize right side window Ctrl+Shift-B, Command+Shift-B => input[text] Show bing result for the input on the right side Ctrl+Shift-O, Command+Shift-O => input[text] Open File by name Ctrl+Shift-Z, Command+Shift-Z => input[text] Execute terminal commands Ctrl-L, Command-L => input[text] Configure which page was showed on the right side
