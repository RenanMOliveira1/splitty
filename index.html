Welcome to splitty.

splitty is a simple static file server with a embedded text editor based on ACE. splitty is focused to speed up the web development allowing the programmer to use de ACE editor on the left side and see the result on the right side of the same browser tab.



Many files simultaneously

With splitty you can also split your browser tab to open many files!



Open multiple files

Open files in the editor with incremental search.



Browse between opened files

With splitty you can open many files in editor and controls them with simples hotkeys.



Execute Bing search

You can search at the Bing website just putting your terms in the action box and the splitty will open a Bing response on the right side of the tab.



Execute shell commands

You can execute shell commands by input your command in the action box and see the stdio from process.



Define your own shell commands aliases

You can define alias for the shell commands at the shell command box by Pressing Ctrl+Shift-Z or Command+Shift-Z and input your new alias like $ > alias: gs = git status . Press enter and Pahhhhh you have an new alias. To use it you must open the shell command box and type your new alias like $ > gs


Dependencies

  • Node JS 4+
  • NPM

Installing

Download or clone the code of splitty and then execute the follow steps:

  • $ cd splitty_PATH_DOWNLOAD
  • $ npm install
  • $ npm link

Using

After installation

  • $ cd PathThatYouWantToServeFiles
  • $ splitty
  • Open the browser at http://localhost:8000/_editor

Manual and Hotkeys

    Serve static files
    $ splitty port=8000
    After start all the files in directory can be 
    accessed in the browser
    For example, if you have a file index.html at the root you 
    can access http://localhost:8000/index.html

    Open splitty Editor
    $ splitty
    Open browser at http://localhost:8000/_editor
